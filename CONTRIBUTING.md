# Contributing to Talastock ETL Data Platform

Thank you for your interest in contributing! This project is a learning-focused data engineering platform.

## 🎯 Ways to Contribute

### 1. Report Issues
- Found a bug? Open an issue
- Have a suggestion? Open an issue
- Documentation unclear? Open an issue

### 2. Improve Documentation
- Fix typos
- Add examples
- Clarify explanations
- Add more SQL queries

### 3. Add Features
- New data generators
- Additional ETL pipelines
- More data quality checks
- Dashboard visualizations

### 4. Share Your Learning
- Write blog posts about using this project
- Create video tutorials
- Share on social media

## 🚀 Getting Started

1. **Fork the repository**
2. **Clone your fork**
   ```bash
   git clone https://github.com/YOUR_USERNAME/talastock-etl-data-platform.git
   cd talastock-etl-data-platform
   ```

3. **Create a branch**
   ```bash
   git checkout -b feature/your-feature-name
   ```

4. **Make your changes**
   - Follow the existing code style
   - Add comments where helpful
   - Update documentation if needed

5. **Test your changes**
   - Run the ETL pipeline
   - Check data quality
   - Verify documentation

6. **Commit and push**
   ```bash
   git add .
   git commit -m "Add: your feature description"
   git push origin feature/your-feature-name
   ```

7. **Open a Pull Request**
   - Describe what you changed
   - Explain why you changed it
   - Link any related issues

## 📝 Code Style

### Python
- Follow PEP 8
- Use meaningful variable names
- Add docstrings to functions
- Keep functions focused and small

### SQL
- Use uppercase for keywords (SELECT, FROM, WHERE)
- Indent for readability
- Add comments for complex queries
- Use meaningful aliases

### Documentation
- Use clear, simple language
- Include examples
- Add screenshots where helpful
- Keep it beginner-friendly

## 🎓 Learning Focus

This project is designed for learning. When contributing:
- Prioritize clarity over cleverness
- Add explanatory comments
- Include learning resources
- Make it accessible to beginners
